# AI Company — منظومة تطوير البرمجيات بالذكاء الاصطناعي

منظومة متعددة الوكلاء (`multi-agent`) لبناء تطبيقات ومواقع انطلاقًا من مواصفات مكتوبة، عبر لوحة تحكم على الويب.

كل الأدوات مفتوحة المصدر و `self-hosted`، تعمل على خادم واحد، وتُدار من المتصفح بأقل استخدام ممكن للترمينال. التكلفة التشغيلية الوحيدة هي استدعاءات الـ `API` لمزوّد نماذج الذكاء الاصطناعي.

## التشغيل

المنظومة كلها تُسلَّم جاهزة في هذا المستودع. على الـ `VM`:

```bash
git clone <repo-url> && cd AI-company && ./install.sh
```

السكربت يثبّت ويشغّل كل الخدمات تلقائيًا. بعدها كل شيء من المتصفح.

## المكوّنات

| المكوّن | الأداة | الدور |
|---|---|---|
| البنية التحتية | `Ubuntu` + `Docker` + `Portainer` | تشغيل وإدارة الخدمات من المتصفح |
| إدارة الأسرار | `Infisical` | كل المفاتيح وكلمات المرور في مكان واحد |
| بوابة النماذج | `LiteLLM` | نقطة وصول موحّدة لكل مزوّدي النماذج |
| المبرمج | `OpenHands` | مكوّن داخلي يكتب وينفّذ الكود |
| تنسيق الوكلاء | `CrewAI` | باحث ← مصمم ← مخطّط ← مراجع |
| تكامل GitHub | خدمة داخلية | ريبو خاص لكل مشروع تلقائيًا |
| لوحة التحكم | واجهة ويب موحّدة | إدخال المشاريع ومتابعتها |

## خط العمل (Pipeline)

```
فكرة/مواصفات
   ↓
باحث      → تحليل + مصادر + نقاش
   ↓
مصمم      → mockup فعلي
   ↓
مخطّط      → خطة و todo مرتّبة
   ↓
OpenHands → تنفيذ الكود خطوة بخطوة
   ↓
مراجع     → فحص الأمان والجودة
   ↓
مراجعة بشرية نهائية
   ↓
ريبو GitHub خاص بالمشروع
```

> ملاحظة: كل مشروع تنتجه المنظومة يُحفظ على GitHub في ريبو مستقل، وليس على السيرفر. السيرفر لتشغيل المنظومة فقط.
